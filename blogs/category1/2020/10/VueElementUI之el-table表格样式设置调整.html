# Vue Element UI el-table 表格样式设置调整

有时候表格中并非全部都是文字，然后影响了自动行高，下面将进行表格行高及单元格的padding修改：

<div class="table_container"><el-table
:data="tableData"stripe
v-loading="loading"width="100%"
height="500”
border
: header-cell-stvle="headStyle”
: row-style="iheight: " 10px'}”
:cell-style=“{padding: '5px 0'}”

# 设置行的样式:

:row-style="{height: '50px'}"

设置单元格的样式:

:cell-style="{padding: '5px 5px'}"

# 也可以使用函数回调方式设置:

# 方式一:

直接使用style

:header-cell-style="headStyle"

//methods
//参数按这个写就行,<el-table>中不需要写
headStyle({row,column,rowIndex,columnIndex}){
	return 'text-align:center';//直接写好css样式
},
tableHeaderStyle({row,column,rowIndex,columnIndex}){
      return 'background:#fafafa;color:#000'
},
cellStyle({row,column,rowIndex,columnIndex}){
   if(columnIndex == 1){
     return 'color:red'
   }
}

# 方式二:

使用class类名称

:header-cell-class-name="headStyle"

//methods
headStyle({row,rowIndex}){
	return 'table-th';//调用CSS里写好的类名
}

//css
.table-th{
	text-align:center;
	height:40px;
	padding:0;
}

# 其他方式

修改头部背景和字体颜色

.el-table th{background: #344067;color: #fff;font-weight: 500;}
1

修改表头字体颜色

.el-table thead {color: #8EB7FA;}
1

修改头部单元格边框颜色

.el-table th.is-leaf {border: 1px solid #114696}
1

修改【奇数】行背景

.el-table tr{background: #1f284c;}
1

修改【奇数】行单元格边框

.el-table tr td{border: 1px solid #114696}
1

修改【偶数】行背景和单元格边框颜色

.el-table--striped .el-table__body tr.el-table__row--striped td {background: #1f284c;border: 1px solid #114696}
1

修改表格最底部颜色

.el-table::before{background:#1f284c;z-index: 0;}
1

修改表格内容部分字体颜色

.el-table{color: #fff;}
1

修改表格无数据背景，字体颜色

.el-table__empty-block{background: #16203C;}
.el-table__empty-text{color: #ccc}
1
2

修改表格鼠标悬浮hover背景色和字体样式

.el-table--enable-row-hover .el-table__body tr:hover>td {background-color: #114695;color: red;}
1

修改表格中头像图片垂直居中显示

.el-avatar{vertical-align:middle;}
