# JSON5的使用

json文件如果存在注释，文件和编辑器都会报红， 提示有错误， 有了json5， 就可以在json文件上写注释了，json5还有啥用? 这里简单介绍一下:

  1. 对象 Objects中key值允许没有双引号.

  2. 允许有多余的逗号结尾.

  3. key值可以使用单引号包裹.

  4. 数组 Arrays 中允许有对于的逗号结尾.

  5. 允许使用单引号包裹字符串.

  6. 字符串可以换行，多行.

  7. 字符串允许使用转义符.

  8. 数字可以使用十六进制表示.

  9. 数字允许使用小数点开头或结尾.

  10. 数字可以使用正无穷大/负无穷大.

  11. 数字前面可以有一个+号.

  12. 支持单行注释，多行注释.

  13. 允许多余的空白符.

    更多用法可以参考官网 (opens new window)

使用vscode编写代码，所以就直接安装一下JSON5扩展， 直接搜索一下 json5， 选择 JSON5 syntax 安装.

// testJSON5.js

const fs = require('fs');
const path = require('path');
const JSON5 = require('json5');
//读取json文件
function getJsonFile(filePath) {
    //读取指定json文件
    var json = fs.readFileSync(path.resolve(__dirname，filePath)， 'utf-8');
    //解析并返回
    return JSON5.parse(json);
}
var json = getJsonFile('./userInfo.json5');
