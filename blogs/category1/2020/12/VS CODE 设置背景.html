VS CODE 设置背景

先来张效果图

开始设置

打开扩展(Ctrl+shift+X),搜索插件 background-cover ,安装它

安装后根据提示操作,重启vscode

插件的使用:

设置图片及透明度,点击vscode底部的状态栏,右边倒数第二个图标,鼠标放上去会提示切换背景图 点击后会显示对应的操作面板

根据需要进行设置即可,简单操作

可以直接选择添加图片目录,这样就可以放多张图片, 然后可以开启自动更换功能

设置完成后会提示重启vscode, 重启就行

重启vscode后会提示 code安装似乎损坏,请重新安装, 这是vscode检测到配置被修改了, 顶部也会显示 不受支持四个字

解决方法:

安装一个扩展: Fix VSCode Checksums

打开命令面板 ctrl+shift+P

输入Fix Checksums: Apply

回车后重启vscode即可
