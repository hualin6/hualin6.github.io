axios安装使用及axios跨域设置

MarkLin 2020-08-09 Vue

# 安装

npm install --save axios
1

# 配置 main.js文件

//导入axios模块
import axios from 'axios'

//配置axios的基础地址(根据实际地址设置)
axios.defaults.baseURL = 'http://127.0.0.1:3000'

//在vue的原型上扩展
Vue.prototype.axios = axios;

查看是否安装成功

打开项目根目录下的package.json文件,查看"dependencies"内是否有"axios": "^0.19.2",

有即代表安装成功, "^0.19.2" 是axios的版本号

至此完成安装与配置

可以在相应.vue的mounted钩子函数中发送请求

此时直接运行脚手架可能出现以下报错 :

 

GET http://127.0.0.1:3000/ nte::ERR_CONNECTION_REFUSED

 

Uncaught(in promise) Error:Network Error

原因: 未启动WEB服务器

启动WEB服务器即可(node app)

# 附Node服务器创建

# 创建Node服务器

脚手架在发送请求时,指定的协议名称为HTTP,所以必须在Node服务器上安装express框架:

npm install --save express
1

安装成功后,在服务器的根目录下创建app.js，示例代码如下：

//引入Express模块

const express = require('express');

//创建Express实例

const server = express();

//指定服务器的监听端口号

server.listen(3000);

脚手架的端口号与服务器端端口号不一致导致的跨域错误

# axios跨域设置

在Node服务器上安装CORS模块且进行配置

安装:

npm install --save cors
1

配置: 继续在app.js中添加一下代码

# origin内的地址是重点（填客户端地址）

//引入CORS模块
const cors = require('cors');

//将CORS作为Server的中间件使用
server.use(cors({ origin:['http://127.0.0.1:8080','http://localhost:8080']}))

# API调用

例: 服务器上创建 /category的API ,请求方式为get ,其结构如下

server.get('/category',(req,res)=>{
  //操作语句
});

此时需要在当前 /category/的API地址中获取数据库中的a表数据, 所以还需要安装MySQL模块,同时进行相关配置:

安装MySQL模块

npm install --save mysql
1

配置MySQL模块(偷懒就继续在app.js中写)

//引入MySQL模块
const mysql = require('mysql');

//创建MySQL连接池
const pool = mysql.createPool({
    //数据库服务器地址
    host:'127.0.0.1',
    //数据库用户名
    user:'root',
    //数据库用户密码
    password:'',
    //数据库服务器端口号
    port:3306,
    //数据库名称
    database:'x',
    //编码方式
    charset:'utf8',
    //连接限制
    connectionLimit:15
});

在后台/category请求API中，将获取数据表的记录返回到客户端，示例代码如下：


//获取所有文章分类信息的API
server.get('/category',(req,res)=>{
    //SQL查询语句
    let sql = 'SELECT id,category_name FROM a';
    //执行SQL查询语句
    pool.query(sql,(err,results)=>{
        if(err) throw err;
        //响应到客户端的信息
        res.send({message:'查询成功',code:1,results:results});
    });
});

# 在客户端接收并显示显示信息

客户端发送请求

mounted(){
    this.axios.get('/category').then(res=>{
        //暂时打印一下,看下是否接收到了数据
    	console.log(res.data.results);
    });
}

获得的数据最终是要显示在网页上的,所以将获得的数据存储在一个data变量category中,然后再通过v-for指令进行循环输出就可以了

data(){

		//用于保存服务器返回的信息
		category:[]
	}

mounted(){
    this.axios.get('/category').then(res=>{
        this.category = res.data.results;
    });
