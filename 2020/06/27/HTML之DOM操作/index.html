HTML之DOM操作

本例使用简单的输入框和按钮实现将用户输入的内容显示出来


1.创建输入框/按钮/显示区域

创建输入框 :

< input type="text" id="int1" value="" />

创建按钮 :

< button onclick="show()">显示< /button>

创建显示区域 :

< div id="d1">这里是显示区域< /div>

2.给输入框和显示区域设定id，给按钮设置一个点击事件，点击时调用一个函数



3.写函数

function show(){

//声明变量获取输入元素对象

var inp=document.getElementById("int1");

//声明变量获取显示区域元素对象

var obj=document.getElementById("d1");

//使用对象.innerHTML来操作显示区域，将输入对象的的值赋给显示区域对象

obj.innerHTML=inp.value;

}



4.函数需在head中并在 < script>函数放这里< /script> 中



简单的说就是设置一个点击按钮事件,调用函数方法,函数中写好dom操作对象,并用对象点出属性
